For both HDRP and URP templates you need to use Unity version 2020.3.0 or newer. 

For more information about HDRP and URP, please read the online documentation: https://docs.google.com/document/d/1JuJERUs9bHl0VlrwWS2Npzt2NnEEhCTISLQhiKrBZ54

HDRP - HOW TO USE:
1. Create a new empty project using HDRP Template.
2. Go to Windows > Package Manager and select High Definition RP.
	- For Unity 2023.2.0 or for a newer version of unity, update the High Definition RP to version 16.0.4 or above.
	- For Unity 2022.3.0, update the High Definition RP to version 14.0.4 or above.
	- For Unity 2021.3.0, update the High Definition RP to version 12.1.6 or above.
	- For Unity 2020.3.0, update the High Definition RP to version 10.3.2 or above.

3. Import Procedural Sky (Windows > Package Manager and select High Definition RP)
4. Import the Winter Environment package in the project.
5. Import the HDRP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 

Note: 
For Unity 2023.2.0 or newer, HDRP version 16.0.4 or above - You need to use HDRP template 15.0.6.unitypackage
For Unity 2022.3.0, HDRP version 14.0.7 or above - You need to use HDRP template 14.0.4.unitypackage
For Unity 2021.3.0, HDRP version 12.1.6 or above - You need to use HDRP template 12.1.0.unitypackage
For Unity 2020.3.0, HDRP version 10.3.2 or above - You need to use HDRP template 10.2.2.unitypackage

--------------------------------------------------------------------------------

URP - HOW TO USE:
1. Create a new empty project using URP Template.
2. Go to Windows > Package Manager and select Universal RP.
	- For Unity 2023.2.0 or for a newer version of unity, update the Universal RP to version 16.0.4 or above.
	- For Unity 2022.3.0, update the Universal RP to version 14.0.7 or above.
	- For Unity 2021.3.0, update the Universal RP to version 12.1.6 or above.
	- For Unity 2020.3.0, update the Universal RP to version 10.3.2 or above.

3. Import the Winter Environment package in the project. 
4. Import the URP Template from ANGRY MESH > Nature Pack - Winter Environment > SRP. 

Note: 
For Unity 2023.2.0 or newer, URP version 16.0.4 or above - You need to use URP template 15.0.6.unitypackage
For Unity 2022.3.0, URP version 14.0.7 or above - You need to use URP template 12.1.0.unitypackage
For Unity 2021.3.0, URP version 12.1.6 or above - You need to use URP template 12.1.0.unitypackage
For Unity 2020.3.0, URP version 10.3.2 or above - You need to use URP template 10.2.2.unitypackage
